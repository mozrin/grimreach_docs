TITLE: Grimreach Workspace – Phase 001: Folder Structure and File Scaffolding

INTENT
Create the full folder structure and file scaffolding for the Grimreach workspace.  
This includes four Dart/Flutter projects:

1. grimreach_client – Flutter + Flame client
2. grimreach_server – Dart backend server
3. grimreach_census – Dart service for player/world analytics
4. grimreach_api – Shared Dart package for protocol, models, and message schemas

No code is required in this phase.  
All files must exist and be empty unless otherwise specified.  
All names and paths must match exactly.
